# exe-simple
простая утилитка для запуска других файлов с параметрами (самописный аналог [proxy.exe](https://web.archive.org/web/20131212003408/http://www.northcode.com/blog.php/2007/08/07/Conquering-FSCommand-EXEC-Part-1-Proxy), [runapp](https://github.com/futurist/runapp) и подобного)

// .ini имеет приоритет перед .bat

// если рядом есть .ini файл, то считываются значения ключей в секции [General]:

// prog - путь до запускаемого файла (exe, bat, cmd, др.),

// args - аргументы для запускаемого файла (спецсимволы типа &^*()/\ должны экранироваться),

// wdir - рабочая папка запуска запускаемого файла (если значение в .ini пустое или wdir вовсе не определено, то принимается рабочей папки значения prog),

// show - видимость окна запускаемого файла - значения hide, min, max (если значение в .ini пустое или show вовсе не определено, то принимается shownormal),

// если .ini файла рядом с .exe не наблюдается, скрыто запускается .bat файл с тем же именем

// имена файлов .bat, .ini должны совпадать с исполяемым

// SimpleIni.h (https://github.com/brofield/simpleini) только для .GetValue

// в конце переменные на всякий случай
